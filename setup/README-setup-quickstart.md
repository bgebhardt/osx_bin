# Quick Start

Short version of what's in README-setup.md

This is currently a scratch space for my mac setup.

1. follow initial settings
2. set up git and gh
3. login to gh
4. clone bin
5. install eza and zoxide
6. brew bundle install -v --file=setup/homebrew/Brewfile.minimum
7. ~/bin/setup/macOS/mac-defaults.sh
8. set up 1Password
9. set up ssh key for mac in 1password
10. install Tailscale
11. install Rosetta - /usr/sbin/softwareupdate --install-rosetta --agree-to-license
12. install Brewfile.homeserver
13. install Brewfile.desktop
14. install Brewfile.ai
15. install Brewfile.games
16. Install and setup Microsoft Office
17. Setup OneDrive
18. Setup iDrive
